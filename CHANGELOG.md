# Changelog

All notable changes to the **Projonexa official website** repository are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-05-29

### Added

- Official Projonexa website with 11 pages (Home, About, Services, Projects, Research, Blog, Portfolio, Pricing, Careers, FAQ, Contact)
- Premium UI with dark/light mode, Framer Motion animations, and glassmorphism design
- Brand design system (colors, typography, gradients) per brand guidelines
- SEO: per-page meta tags, Open Graph, Twitter cards, JSON-LD structured data
- Static sitemap and robots.txt
- Comprehensive README with services, technology stack, and founder information
- Community health files: LICENSE (AGPL-3.0), Code of Conduct, Contributing, Security policy
- GitHub templates: bug reports, feature requests, pull request template
- CI workflow for build verification on push and pull requests
- Dependabot configuration for npm and GitHub Actions updates
- Documentation: Architecture, Brand Guidelines, SEO Strategy, GitHub Repository Setup

### Tech Stack

- React 18, TypeScript, Vite 5, Tailwind CSS 3, Framer Motion, React Router
